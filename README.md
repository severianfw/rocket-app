# 🚀 Rocket App

A modern Android application that displays SpaceX rockets and the next upcoming launch with a live countdown.

This project was built as a **learning and portfolio project** to explore **Modern Android Development**, experiment with new Android technologies, and practice clean architecture and polished UI.

---

## ✨ Features

- 🚀 Rocket list and detail
- 🎬 Shared Element transition between screens
- ⏳ Live countdown to the next rocket launch (in progress)
- 🌐 Data from SpaceX public API

---

## 🏗 Architecture

Built following **Modern Android Development (MAD)** principles.
```
UI (Jetpack Compose)
│
ViewModel
│
Repository
│
Remote API
```
Key concepts:
- MVVM
- Unidirectional data flow
- Separation of concerns

---

## 🕹️ Tech Stack

- **Kotlin**
- **Jetpack Compose**
- **MVVM**
- **Coroutines & Flow**
- **Retrofit**
- **Coil**
- **SpaceX API**

---

## 🎯 Project Goals

- Practice **modern Android architecture**
- Explore **new Android development tools**
- Experiment with **UI animations and Compose**
- Build a **clean and maintainable Android project**
